Présentation

Le concept

Transformer un tirage au sort ordinaire en événement mémorable qui anime toute la salle.

Le problème

Aujourd'hui dans les casinos, un tirage au sort se résume à un papier tiré dans une urne ou un affichage sur écran. Le moment manque de théâtralité — les joueurs ne regardent pas, l'émotion collective est absente.

Casino Light Node change ça. Chaque machine à sous est équipée d'une bande LED discrète. Au moment du tirage, les lumières s'allument une par une, créent un effet de roulette dans toute la salle, ralentissent progressivement... jusqu'à ce qu'une machine s'illumine en or.

Ce que ça apporte
🎰
Expérience immersive
La lumière attire l'attention de toute la salle. Même les joueurs non participants regardent.
📱
Contrôle mobile
Le responsable lance le tirage depuis son téléphone en une seconde, sans ordinateur dédié.
⚡
Non intrusif
Bande LED sous la machine. Aucune modification de la machine elle-même. Pas d'homologation nécessaire.
🎨
Personnalisable
Couleurs, ambiances, nombre de gagnants — tout se configure depuis l'interface.
Les deux composants
🎛
Contrôleur — 1 par salle
Crée le réseau WiFi · Héberge le backoffice · Tire le gagnant · Envoie les commandes
💡
Light Node — 1 par machine
Se connecte au WiFi · Reçoit les commandes · Pilote la bande LED · Envoie un ping toutes les 5s
Le déroulement d'un tirage
1
Ouverture du backoffice
Connecté au WiFi CASINO_CTRL, le responsable accède à l'interface via 192.168.4.1 dans son navigateur.
2
Choix du mode et lancement
Roulette classique, multi-gagnantes ou plage personnalisée. Un seul bouton suffit.
3
Animation dans toute la salle
Flash blanc de départ, roulette qui tourne sur les machines, ralentissement progressif, suspense sur les dernières machines.
4
La machine gagnante s'illumine
Flashs lumineux, puis pulsation douce infinie. En multi-gagnantes, chaque gagnante garde sa couleur unique.
5
Célébration
En mode multi-gagnantes, toutes les machines gagnantes s'animent en arc-en-ciel pendant 3 secondes.

Modes de tirage

Trois façons d'animer votre salle.

Mode 1 — Roulette classique

La roulette tourne sur toutes les machines, une seule gagnante. Animation or avec pulsation jusqu'au prochain tirage.

Mode 2 — Multi-gagnantes

De 2 à 6 gagnantes successives. Chaque roulette désigne une machine qui s'illumine dans une couleur et garde sa couleur pendant que les suivantes se jouent. À la fin, toutes les gagnantes célèbrent ensemble en arc-en-ciel.

Ambiances lumineuses

Votre salle vit au rythme de vos événements entre les tirages.

Entre les tirages, les machines ne sont pas éteintes. Le système propose 6 modes d'ambiance pour maintenir une atmosphère lumineuse cohérente avec chaque événement.

🌙
Veille
Bleu nuit très discret (20, 20, 40).
💗
St Valentin
Fondu doux rose (220,20,100) vers rouge (200,0,0).
🎄
Noël
Fond rouge avec scintillements blancs aléatoires.
🎃
Halloween
Fondu violet (148,0,211) vers orange (255,50,0).
🌈
Arc-en-ciel
Spectre complet HSV réparti sur les bandes.
🎨
Couleur libre
Choisissez n'importe quelle couleur depuis le sélecteur. Adaptable à votre charte.
Ambiances par zone

Chaque zone peut avoir sa propre ambiance indépendante. La commande NODE_AMBIANCE cible précisément les nodes d'une zone sans affecter les autres.

Chaque machine peut également recevoir une couleur individuelle indépendante — pour signaler une machine VIP, une zone premium ou une machine hors service.

Règles de communication
  • Le contrôleur envoie des commandes UDP broadcast vers tous les nodes simultanément
  • Chaque node envoie un ping UDP vers le contrôleur toutes les 5 secondes
  • Le contrôleur déclare un node hors ligne après 15 secondes sans ping
  • Le backoffice web se connecte au contrôleur via HTTP sur 192.168.4.1
  • Le backoffice se rafraîchit automatiquement toutes les 5 secondes (polling)

Configuration des Light Nodes

Premier démarrage

À la livraison ou après un reset, un node démarre en mode configuration :

1
LEDs orange
Le node démarre en mode configuration, les LEDs s'allument en orange.
2
Connexion au réseau NODE_XXXXXX
Un réseau WiFi unique apparaît avec le suffixe MAC. Mot de passe : config123.
3
Saisie du numéro
Ouvrir 192.168.4.1, saisir le numéro de machine (1 à 999), valider.
4
Confirmation
LEDs vertes 2 secondes → veille bleue → node prêt et connecté au CASINO_CTRL.
Procédure sur iPhone
⚠️
Étape obligatoire entre chaque nodeSans cette procédure, l'iPhone utilise le cache du réseau précédent et deux nodes reçoivent le même numéro.

Configurer le node
Attendre le redémarrage (LEDs vertes puis bleues).
2
Oublier le réseau
Réglages → WiFi → appuyer sur (i) à côté de NODE_XXXXXX → Oublier ce réseau.
3
Node suivant
Se connecter au prochain NODE_YYYYYY et taper manuellement 192.168.4.1.

Adressage des nodes

Chaque node est identifié par un numéro entier de 1 à 999 stocké en mémoire flash. Ce numéro correspond au numéro de la machine à sous physique dans la salle.

En mode configuration, le réseau WiFi du node est nommé NODE_XXXXXX où XXXXXX est le suffixe de l'adresse MAC — garantissant l'unicité même avec des dizaines de nodes.
